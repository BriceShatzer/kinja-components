# MetaInfo

Meta info element what renders `<StoryTypeLabel>` for Story Type, Category, and Subcategory and `NativeShareLabel`.

<!-- STORY -->

## Props

### post

Type: _PostJSON_

Post Object.

### blog

Type: _Blog_

Blog object where the post will appear.

## Sample usage

```javascript
<MetaInfo blog={blog} post={post} />
```

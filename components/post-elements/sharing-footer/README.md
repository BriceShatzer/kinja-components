## Props

| Name | Type | Required | Default value | Description
|------|------|----------|---------------|------------
| facebookShareUrl | `string` | x |  | Facebook sharing url for post
| twitterShareUrl | `string` | x |  | Twitter sharing url for post

## Example


# Running the Unit Test Suite and Linting your code ![ready](status-images/ready.svg)

We write unit and integration tests with Jest.

- `yarn test` run the jest test suite.
- `yarn lint` checks the code style. It's also important to check that your code follows our guidelines.
- `yarn flow` checks that you're using the right types. Type safety is also a priority for us.
- `yarn stylelint` linter that helps us avoid errors and enforce conventions in our styles.
